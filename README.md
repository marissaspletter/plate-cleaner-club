# Plate Cleaner Club Website V3

This version includes:
1. Featured Restaurant Partners
2. Restaurant Success Stories
3. Redesigned Google Review Impact section
4. Live Social Feed placeholder with direct Instagram/TikTok links
5. Interactive Food Map

## Publish with GitHub Pages
1. Upload all files and folders in this zip to the root of the `plate-cleaner-club` GitHub repo.
2. Go to Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /root.
6. Save.

Your site should publish at:
https://marissaspletter.github.io/plate-cleaner-club/
